<action>
Create src/actions/update-course.ts. Add "use server" at top.

Export async function updateCourse(courseId: string, formData: FormData): Promise&lt;{ success: true } | { error: string }&gt;

Steps:
1. verifySession() — if no session, return { error: "Unauthorized" }
2. Fetch the course by courseId — if course.creator_id !== session.userId, return { error: "Forbidden" }
3. Extract title (string), description (string), visibility ("public"|"private") from formData
4. Update the courses row: title, description, visibility
5. Return { success: true }
</action>
<acceptance_criteria>
- src/actions/update-course.ts exports updateCourse
- Ownership check: only course.creator_id === session.userId may update
- Updates title, description, and visibility columns
- Returns { error: "Forbidden" } for non-owners
</acceptance_criteria>
</task>
