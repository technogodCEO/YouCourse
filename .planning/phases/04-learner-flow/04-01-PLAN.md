<action>
Create src/actions/submit-quiz.ts. Add "use server" at top.

Export async function submitQuiz(lessonId: string, answers: number[]): Promise&lt;{ passed: boolean; score: number; correctAnswers: number[] }&gt;

Steps:
1. verifySession() — if no session, return { passed: false, score: 0, correctAnswers: [] }
2. Fetch all questions for lessonId ordered by position
3. Score: count how many answers[i] === questions[i].correct_index. score = correct / total * 100.
4. passed = score >= 70
5. If passed: update user_progress set passed_quiz=true, completed_at=now() where user_id=session.userId AND lesson_id=lessonId
6. Return { passed, score: Math.round(score), correctAnswers: questions.map(q => q.correct_index) }
</action>
<acceptance_criteria>
- submitQuiz exports with signature (lessonId: string, answers: number[]) => Promise&lt;{passed, score, correctAnswers}&gt;
- 70% threshold: score >= 70 sets passed=true
- Only updates user_progress when passed=true
- Returns correctAnswers array so client can highlight right/wrong answers
</acceptance_criteria>
</task>
