Create src/actions/generate-course.ts. Add "use server" at top.

Export async function generateCourse(formData: FormData): Promise&lt;{ courseId: string } | { error: string }&gt;

Steps:
1. Call verifySession() — if no session, return { error: "Unauthorized" }
2. Extract topic (string) and lengthPreset ("quick"|"standard"|"long") from formData
3. Insert a new courses row with status="generating", visibility="private", creator_id=session.userId, topic, title=topic (placeholder), length_preset=lengthPreset. Capture the new courseId.
4. Call generateCurriculum(topic, lengthPreset) to get lessonTopics[].
5. Insert all lesson stub rows first (sequential, to preserve position order), capturing each lessonId. Each row: position=index, topic=lessonTopic, transcript_status="pending".
6. Process all lessons in parallel using Promise.all. For each lesson:
   a. Call searchYouTubeVideo(lessonTopic). If null, update lesson transcript_status="unavailable" and return.
   b. Update the lessons row: youtube_video_id, video_title, video_duration_seconds.
   c. Call fetchTranscript(videoId). If null, update lessons.transcript_status="unavailable" and return.
   d. Update lessons: transcript_cached=transcript, transcript_status="fetched".
   e. Call generateQuestions(transcript, lessonTopic). Insert all 5 questions rows linked to lessonId.
7. Update courses.status="ready", courses.title="{topic} — Generated Course".
8. Return { courseId }.

Wrap the entire pipeline in try/catch — on error, update courses.status="error" and return { error: message }.
</action>

- On any error, courses.status is updated to "error"
