# YouCourse

## What This Is

YouCourse is a web app that lets anyone build and take structured learning courses from YouTube videos. Creators sequence YouTube videos into courses; AI generates comprehension questions at creation time. Learners watch each video, answer questions (must pass to proceed), and progress through the course sequentially. The app is mobile-responsive from day one.

## Core Value

A student finishes a course with provable comprehension — not just views, but verified understanding — because every video is gated by questions they had to pass.

### Active

- [ ] Authenticated user can create a course by adding YouTube videos in sequence

### Out of Scope

- Native mobile app — web with responsive design is sufficient for v1; native comes later if needed
- Manual question authoring — everything AI-generated for v1; official curated courses are a future milestone
- Paywall / monetization — private course paywalling is planned but deferred to a later milestone
- Live video or non-YouTube sources — YouTube API only for v1

## Context

- YouTube Data API v3 is the video source; API quota must be conserved — videos and metadata are fetched at course creation and cached, not on each student view
- Questions are generated by an LLM at course creation using video transcripts (YouTube captions); subsequent "more questions" requests reuse the already-cached transcript/metadata
- The scoring gate (must pass to proceed) is the core learning mechanic — pass threshold TBD (likely 70%)
- Public/private visibility is user-controlled at publish time; private courses are eventually paywalled but billing is out of scope for v1

## Constraints

- **API Quota**: YouTube Data API calls minimized — fetch once at course creation, cache everything
- **LLM API Calls**: Questions generated in batch at creation; additional questions use cached context only
- **Platform**: Web app first, mobile-responsive; no native app in v1
- **Auth**: Required for all course creation; browsing public catalog may be anonymous
